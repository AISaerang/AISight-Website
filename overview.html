Kurikulum & Modul

Struktur pembelajaran dari dasar hingga siap kerja. Modul disesuaikan dengan paket: Specialize, Regular, dan Advance.

Pre-Test

  • Multiple-choice Pre-Assessment (Quiz)

Excel Fundamentals

  • Core spreadsheet skills, cleaning, lookup, pivot
  • Shortcuts & productivity tips

SQL for Data Analysis

  • SELECT/JOIN/AGG, filtering, window functions (dasar)
  • Hands-on query untuk analisis

Power BI Dashboard

  • Power Query, data model & relationships
  • DAX dasar–menengah, measures
  • Dashboard & storytelling

Python for Analytics

  • Pengenalan Python untuk analitik
  • Pandas untuk transformasi & analisis

Web BI Portfolio

  • Publikasi dashboard & pembuatan portfolio web
  • Best practice presentasi & review

Post-Test

  • Evaluasi & umpan balik

Paket

  • Specialize: Power BI, Web BI Portfolio
  • Regular: Basic Excel, Power BI, Web BI Portfolio
  • Advance: Excel, SQL, Power BI, Python, Web Portfolio

Durasi per Modul

  • Basic Excel — 8 jam
  • Basic SQL — 8 jam
  • Power BI — 10 jam
  • Basic Python — 10 jam
  • Web BI Portfolio — 6 jam
